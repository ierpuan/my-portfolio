# 🖤 Portfolio Iqbal

Website portfolio full-stack dengan React.js, Express.js, MySQL, Tailwind CSS, dan Alpine.js.

## Fitur Lengkap

- ✅ **Hero** — Typewriter effect, status badge, CTA
- ✅ **About** — Bio, stats, terminal box, social links
- ✅ **Skills** — Animated skill bars, tab per kategori
- ✅ **Projects** — Card grid, tech stack tags, GitHub/demo links
- ✅ **Contact** — Form dengan validasi, data tersimpan ke MySQL
- ✅ **Blog** — Daftar artikel, filter tag, detail artikel, view counter
- ✅ **Resume** — Download CV dari server
- ✅ **Responsive** — Mobile-first design
- ✅ **Dark theme** — Minimalis monochrome
- ✅ **Scroll reveal** — Animasi masuk saat scroll
- ✅ **Rate limiting** — Proteksi form spam
- ✅ **Alpine.js** — Mobile menu toggle
